# react-native-number-ticker
A simple, lightweight and easy-to-use ticker component for natively animating smooth ticker numbers.

![](preview-gif.gif)


## Example usage: 

```javascript
<NumberTicker
   textSize={40}
   number={34.98}
   duration={1500}
   textStyle={{fontWeight: 'bold', color: 'white'}}
/>
```
